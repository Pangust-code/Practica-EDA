# EDA Titanic - Carpeta separada

Esta carpeta contiene un analisis exploratorio completo del dataset Titanic, incluyendo un datasheet en formato Markdown.

## Dataset

- Fuente: https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv
- Variable objetivo: `Survived` (0 = no sobrevivio, 1 = sobrevivio)

## Archivos principales

- `0.2.AnalisisExploratorio_Titanic.py`: script principal de EDA
- `Informe_Guia_Titanic.md`: guia para redactar el informe final
- `Presentacion_Guia_Titanic.md`: guion para exposicion en PDF/PPTX
- `output/DATASHEET_TITANIC.md`: datasheet final del dataset
- `output/REPORTE_FINAL_TITANIC.md`: informe final listo para entrega
- `output/*.csv`: tablas de soporte
- `output/plots/*.png`: visualizaciones

## Ejecucion

Desde la raiz del proyecto:

```powershell
.\.venv\Scripts\python.exe .\2.EDA_Titanic_Datasheet\0.2.AnalisisExploratorio_Titanic.py
```
